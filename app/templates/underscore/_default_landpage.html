Flask-ChatBot

This is a simple Chatbot trained with a few small to medium-size datasets (Messenger logs, Cornel Movie Dialogs Corpus). The backend is built with Flask. You can get the source code on my GitHub repository and you can also check the tutorial of both training this Chatbot and of depoying it.
